# The Letter Griddle Cafe

Cozy vignettes from Griddle Falls paired with word puzzles.

Part of [The Letter Griddle Games](https://lettergriddle.com)

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) to view in browser.

## Deploy on Vercel

Push to GitHub, then import to [Vercel](https://vercel.com).
